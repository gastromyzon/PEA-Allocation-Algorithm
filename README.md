# PEA-Allocation-Algorithm
projet de finance pour le choix d'action sur un pea avec un horizon d'investissement de 5 ans

# Algorithme Quantitatif d'Allocation PEA 

Ce projet propose une solution automatisée pour sélectionner et optimiser un portefeuille boursier européen pour un pea.

##  Fonctionnalités
- **Scoring Multi-Facteurs** : Analyse de la santé financière, de la croissance et de la valorisation.
- **Simulation de Monte Carlo** : Projection stochastique basée sur le Mouvement Brownien Géométrique.
- **Optimisation Gloutonne** : Allocation du capital sous contrainte de budget et de diversification.
- utilisation de la fonction visualisation(budget,liste_pea)
